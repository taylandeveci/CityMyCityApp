## 🚀 Başlatma Talimatları

1. **Terminalde çalıştırın:**
   ```bash
   cd /Users/taylandeveci/Desktop/CityProject/CityMyCityApp
   npx expo start
   ```

2. **QR kodu tarayın:**
   - iOS: Kamera uygulaması ile
   - Android: Expo Go uygulaması ile

3. **Simulator'da test edin:**
   - iOS: `i` tuşuna basın
   - Android: `a` tuşuna basın
   - Web: `w` tuşuna basın
